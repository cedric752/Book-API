# Book Api
APi for managing books, authors and genres.

## Routes


### Books
"GET: /books"
### Protected Routes
"PUT: /books/{id}"
"POST: /books"
"DELETE: /books/{id}"

### Authors
"GET: /authors"
### Protected Routes
"PUT: /authors/{id}"
"POST: /books"
"DELETE: /books/{id}"

### Genres
"GET: /genres"
### Protected Routes
"PUT: /genres/{id}"
"POST: /genres"
"DELETE: /genres/{id}"

### Auth
"POST: /register"
"POST: /login"
"POST: /me"
### Protected Routes
"POST: /logout"
